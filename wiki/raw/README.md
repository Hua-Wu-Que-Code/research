# 原始资料层

这里保存来源材料的索引、摘录或轻量文本版本。

## 子目录

- `articles/`：网页文章、技术博客、官方文档摘录。
- `papers/`：下载的论文 PDF 及其元数据。
- `transcripts/`：需要被 wiki 页面引用的原始会议文字稿或访谈记录。整理过的组会纪要（议程、决策、待办）放 `meetings/`，不放这里。
- `assets/`：图片、图示、外部资源索引；大文件不要提交到 Git。

## 规则

- 原始资料层默认不可变，后续分析应写到知识页。
- 重新 ingest 同一来源时，记录日期和内容变化。
- 不保存密钥、隐私数据或不允许公开的全文材料。
- 受版权保护的 PDF 抽取全文应放在仓库根目录的 `raw/extracted/`，不要放入 `wiki/raw/` 并提交。
