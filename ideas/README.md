# 候选创新点

存放候选研究想法和创新点评估。

## 命名规则

`idea-NNN-简短描述.md`，NNN 从 001 开始，不复用编号。

示例：`idea-001-evidence-selection-bias.md`

## 使用模板

使用 `templates/idea-review.md`。

## 规则

- 每个文件对应一个候选创新点，使用 `templates/idea-review.md`，判断选一种：可作为主贡献、只能作为组件、只能作为诊断信号、不值得继续。
- 推测性想法放这里，不要直接写入 `wiki/`。
- 通过评估的创新点在 `experiments/` 中创建对应实验目录。
